Breast Cancer Mortality Prediction
Project Overview

This project focuses on predicting 10-year breast cancer mortality using machine learning models based on clinical and patient data.

Objective

Identify high-risk patients early

Support data-driven clinical decision-making

Models Used

Decision Tree (Final Model)

Logistic Regression

Random Forest

XGBoost (comparison)

Final Model Performance

Accuracy: ~75%

ROC-AUC: ~0.82

Balanced Precision and Recall

Why Decision Tree

Best overall performance among evaluated models

Strong ability to capture non-linear clinical relationships

Interpretable and suitable for healthcare use cases

Tech Stack

Python, Scikit-learn, Pandas, NumPy

Matplotlib, Seaborn

Jupyter Notebook
